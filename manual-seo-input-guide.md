### 3️⃣ Trainers
**일단 빈 배열로 저장** (6단계에서 처리)

---

## 📝 입력 절차

1. **Sanity Studio → SEO Settings → Create**
2. **Root Page 섹션**: 위 데이터 입력
3. **Centers 섹션**: 
   - "Add item" 클릭
   - Center ID: `wangsimni` 선택
   - 각 페이지별 데이터 입력
   - 대치점, 청담점도 동일하게 반복
4. **Trainers 섹션**: 빈 상태로 저장
5. **Publish 버튼**으로 저장

입력 완료 후 알려주세요! 🚀
